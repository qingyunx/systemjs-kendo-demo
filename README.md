## Installation

```bash
$ cd <project>
```
```bash
$ npm init -y
```
```bash
$ npm install jspm --save-dev
```
```bash
$ jspm init -y
```
```bash
$ jspm install css json text jsx
```
```bash
$ jspm install jquery bootstrap kendo-ui fetch eventemitter3
```
```bash
$ jspm bundle app/main
```
